# Автоматизация планировщика задач Cron (Bash-скрипт)

Скрипт на языке Shell (Bash) для автоматической настройки и инициализации периодических задач в планировщике `cron` в Unix-подобных операционных системах (Linux / macOS).

## Назначение скрипта
Скрипт автоматизирует рутинные задачи системного администрирования бэкенда:
- Автоматическое создание бэкапов баз данных PostgreSQL / MySQL по расписанию.
- Регулярная очистка кэша проекта (`__pycache__`, логи) для экономии места на сервере.
- Периодический запуск фоновых задач, парсеров или скриптов автоматизации.

## Использование

1. Сделайте скрипт исполняемым:
   ```bash
   chmod +x setup_cron.sh
   ```

2. Запустите скрипт для настройки планировщика:
   ```bash
   ./setup_cron.sh
   ```

## Стек технологий
- Shell / Bash scripting
- Linux CLI (crontab, chmod)
